# PharoRustHashMap
Pharo bindings for the thread-safe HashMap in Rust
## Load

```Smalltalk
Metacello new
	baseline: 'RustHashMap';
	repository: 'github://sebastianconcept/PharoRustHashMap/src';
	load.
```
